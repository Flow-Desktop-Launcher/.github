# Flow Launcher – Smart Application Launcher for Windows

Flow Launcher is a fast, extensible, and open-source search tool for Windows. Flow Launcher helps you open apps, search files, run commands, and automate repetitive tasks. Flow Launcher is designed for users who want to replace slow start menus with instant keyboard-driven workflows. Flow Launcher supports plugins for everything from browser history to system commands. Flow Launcher keeps your hands on the keyboard and your mind in focus. Flow Launcher is the modern launcher that grows with your productivity needs.

## About Flow Launcher

Flow Launcher began as a lightweight alternative to built-in Windows search. Today, Flow Launcher is a full-featured productivity engine. Flow Launcher uses a plugin architecture, so you can extend Flow Launcher to control Spotify, search YouTube, manage Docker containers, or interact with ChatGPT. Flow Launcher remembers your habits and prioritizes results accordingly. Flow Launcher works with everything from PowerShell scripts to custom web searches. Flow Launcher respects your privacy – no telemetry, no cloud dependency, no hidden data collection. Flow Launcher is maintained by a community of developers who use Flow Launcher daily. Whether you're a developer, writer, designer, or sysadmin, Flow Launcher adapts to your workflow.

## Getting Started with Flow Launcher

1. Download the latest release of Flow Launcher from the official repository.  
2. Run the installer or portable version of Flow Launcher.  
3. Press `Alt + Space` to open Flow Launcher.  
4. Type a few letters of any app or file – Flow Launcher shows results instantly.  
5. Install plugins inside Flow Launcher using the `pm` command.  
6. Customize hotkeys, themes, and search sources in Flow Launcher settings.  
7. (Optional) Enable Flow Launcher to start with Windows for immediate access.

## Overview of Flow Launcher
![Flow Launcher Dashboard](https://www.flowlauncher.com/_astro/cm_modern.CljlJEm0_ZPaB0.png)

Flow Launcher combines file search, web search, calculator, system commands, and plugin actions into one unified interface. Flow Launcher uses fuzzy search so typos don't stop you. Flow Launcher indexes your Start Menu, installed apps, environment paths, and custom folders. Flow Launcher stays responsive even with hundreds of plugins enabled. Flow Launcher supports dark mode, custom CSS themes, and full Unicode. Flow Launcher is written in C# and .NET for native Windows performance. Flow Launcher can be extended in Python, C#, or any language via standard input/output.

## Key Features of Flow Launcher

- **Instant Search** – Flow Launcher finds apps, files, folders, and settings in milliseconds.

- **Custom Hotkeys** – Assign any key combination to open Flow Launcher or trigger actions.
